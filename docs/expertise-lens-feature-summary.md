# 전문성 렌즈 기능 요약

이 문서는 팀별 전문지식 렌즈 고도화 기능이 실제 화면과 데이터 흐름에 어떻게 반영되었는지 정리합니다.

---

## 1. 목적

전문성 렌즈 기능은 각 팀의 선택과 산출물을 단순한 글쓰기나 점수로 보지 않고, 해당 팀이 가진 기능 전문성과 리더십 판단 기준으로 해석하기 위한 장치입니다.

참가자 화면에는 이론 용어를 과하게 드러내지 않고, 강사용 화면과 리포트에서 해석할 수 있도록 설계했습니다.

---

## 2. 데이터 흐름

```text
팀별 전문지식 렌즈
→ 산출물 입력 기준
→ 산출물 증거 수준 저장
→ 결과 카드 전문성 신호
→ 교육 리포트 전문성 요약
```

## 4. 팀별 전문지식 렌즈

각 팀에는 다음 정보가 있습니다.

- 기능 전문성
- 리더십 관찰 포인트
- 관련 역량
- 좋은 산출물 기준
- 흔한 오판
- 강사용 질문

예를 들어 영업팀은 고객 반응과 신뢰 리스크를 보고, 재무팀은 비용 구조와 투자 효과를 봅니다.

---

## 5. 산출물 증거 수준

산출물 저장 시 다음 네 가지를 기준으로 증거 수준을 기록합니다.

| 기준 | 의미 |
|---|---|
| 판단 근거 | 왜 그렇게 판단했는가 |
| 구체 증거 | 수치, 빈도, 시점, 사례가 있는가 |
| 다음 행동 | 누가, 언제, 무엇을 확인할 것인가 |
| 남는 리스크 | 부담, 불안, 병목, 신뢰, 비용 같은 신호를 다뤘는가 |

증거 수준은 다음 네 단계입니다.

```text
강함 / 충실 / 기본 / 부족
```

---

## 6. 화면 반영

| 화면 | 반영 내용 |
|---|---|
| Team 화면 | 팀별 좋은 산출물 기준, 주의할 오판, 저장된 증거 수준 |
| ResultCard | 전문성 신호, 관련 역량, 팀별 좋은 산출물 기준, 증거 신호 |
| Facilitator Guide | 팀별 기능 전문성, 리더십 관찰, 강사용 질문 |
| Report | 평균 증거 수준, 강한 주차, 취약 주차, 보완 신호 |
| Markdown Report | `팀별 전문성 요약` 섹션 |

---

## 7. 현재 설계 판단

산출물 증거 수준은 현재 최종 판정 점수에 강하게 반영하지 않습니다.

이유는 11개 플레이 라운드 기준의 밸런싱이 이미 적용되어 있기 때문입니다. 먼저 관찰, 피드백, 리포트에 반영하고, 실제 파일럿 결과를 본 뒤 최종 판정에 약하게 반영하는 방식이 안전합니다.
